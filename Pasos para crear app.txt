Para crear esta aplicación web en PHP se necesitarán los siguientes elementos:

 Un servidor web que admita PHP y MySQL.
 Un editor de texto o un IDE para escribir el código PHP.
 Un sistema de gestión de bases de datos MySQL.
 Una biblioteca de correo electrónico para enviar correos electrónicos.
 
Pasos para crear la aplicación:

 Crear la base de datos MySQL con tablas para clientes, productos y pedidos.
 Escribir el código PHP para permitir que los clientes hagan pedidos en línea y para actualizar la base de datos con los detalles del pedido.
 Escribir el código PHP para permitir que los clientes vean el estado de sus pedidos.
 Integrar una biblioteca de correo electrónico para enviar un correo electrónico al cliente con los detalles del pedido.
 Probar la aplicación web para asegurarse de que funciona correctamente.
 Desplegar la aplicación web en un servidor web para que los clientes puedan acceder a ella.

Para hacer que la aplicación sea más segura, se deben tomar medidas como:

 Validar los datos de entrada del usuario para evitar inyecciones SQL y otros ataques.
 Utilizar contraseñas seguras y cifrado para proteger la información del cliente.
 Limitar los privilegios de la base de datos y los usuarios del sistema.
 Realizar copias de seguridad regulares de la base de datos para proteger contra la pérdida de datos.
 Mantener la aplicación y las bibliotecas de terceros actualizadas para corregir errores y vulnerabilidades conocidas.
